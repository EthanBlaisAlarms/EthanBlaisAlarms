Development

The EBA Virtual Fire Panel is built using Clickteam Fusion Developer 2.5+. Clickteam Fusion is a game and application development software, and it's easy to use once you learn how it works. There are a ton of user-created extensions, written in C++, that can add additional features to your applications. Fusion can export in a wide variety of types, most notably EXE, APK, AAB, and HTML5.

With the EBA Virtual Fire Panel, the code is grouped into several groups, such as "Main" which stores the code for the app itself, not for the panel. A few important ones:

  • "Menu" - Stores the code for the application menus.
  • "LED" - Stores the panel's LED coding.
  • "Power and Boot" - Controls the panels main boot sequence and power down events.
  • "Operation" - Stores all of the code for the panels main operations.

Many of these, such as "Operation", also have several sub-groups. For example, "Operation" has "Screens" to control the displays, and "Button Actions" to control the actions that buttons perform.

Version 2 of the EBA Fire Panel has 3594 lines of code! Version 1 had 1397 lines, which just shows how much more advanced version 2 is.
